# 🐢 Turtle Race Game

A fun turtle racing game built using Python Turtle Graphics.

## 📌 Features
- Multiple racing turtles
- User betting system
- Random turtle movement
- Winner announcement

## 🛠 Technologies Used
- Python
- Turtle Module
- Random Module

## 🎮 How to Play
1. Run the program
2. Enter a turtle color to place your bet
3. Watch the race
4. See if your turtle wins!
